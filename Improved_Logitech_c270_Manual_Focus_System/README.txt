# Summary

Thanks to Vecko for recreating each part in step so I could rework the focuser so it uses a STANDARD M3 8mm length screw vs the I have no idea screw from the original. I'm including the Fusion 360 file so you too can make adjustments. 

Basically this makes it so you can easily manually focus you Logitech c270 camera. I've also upgraded this with a wide angle lens so you can see so much more. I use this adapter from amazon and cut out the screw mount and glue it to the frame for easy of use.
